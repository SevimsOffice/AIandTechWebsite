Step 01

Generate the Brand Identity

Ask ChatGPT to create a professional agency-level branding prompt that maintains design principles, typography, color consistency, lighting consistency, premium layouts, and cohesive branding. Then use that prompt to generate all branding visuals.

✓ Upload your logo to ChatGPT as the core brand identity
✓ Paste the Master Brand Identity Prompt below
✓ Replace "MANUBOKU" and the color palette with your brand details
✓ Generate exactly 20 separate images (not a collage)
Master Brand Identity Prompt — Paste into ChatGPT

Create a complete premium brand identity system for a modern brand named "[YOUR BRAND NAME]", using the provided logo as the core identity.

OUTPUT FORMAT (VERY IMPORTANT):

  • Generate EXACTLY 20 separate images
  • Each image must be exported individually (not combined)
  • Do NOT create a collage, grid, sheet, or multi-panel composition
  • Each image must contain ONLY ONE deliverable
  • All 20 images must follow the same visual system and consistency

Generate these 20 images (one per image):

  1. Logo variations (primary, secondary, monochrome, icon-only)
  2. Color palette presentation
  3. Typography showcase
  4. Business card design
  5. Letterhead design
  6. Envelope design
  7. Product packaging mockup
  8. Product close-up (texture + details)
  9. Social media post design
  10. Social media carousel layout
  11. Website homepage mockup
  12. Mobile UI product screen
  13. Presentation slide design
  14. Brand guidelines overview
  15. Minimal product advertisement
  16. Outdoor billboard mockup
  17. Street poster / wall branding
  18. Digital screen advertisement
  19. Office wall branding
  20. Final brand composition (logo + product + tagline)

Art Direction:

  • Premium, modern, minimal, clean
  • Color palette: [your brand colors]
  • Lighting: soft natural studio lighting (slightly warm)
  • Mood: high-end, organic, calm, polished
  • Use subtle shadows and smooth gradients

Design Principles:

Balance · Contrast · Alignment · Proximity · Repetition · Hierarchy · White space · Proportion · Emphasis · Unity · Simplicity · Consistency · Visual flow

CONSISTENCY (critical):

  • Same product design in all images
  • Same color palette across all images
  • Same lighting style across all images
  • Same typography style across all images
  • All outputs must feel like ONE unified premium brand system

Step 02

Generate Social Media Posts

For the social media post, upload your logo as Image 1 and a reference post as Image 2. Then ask ChatGPT to create a completely original premium design inspired by the reference.

✓ Upload your logo as Image 1 (@image1)
✓ Upload a reference post you like as Image 2 (@image2)
✓ Paste the Social Media Post Prompt below
✓ The output should feel like a fresh premium brand post, not a remake
Social Media Post Prompt — Paste into ChatGPT

Use @image1 (my logo) as the primary brand identity and @image2 (reference post) only as structural and layout inspiration.

Create a completely original social media post design that feels unique and not copied.

Requirements:

  • Integrate my logo naturally into the design
  • Extract colors from the logo and build a cohesive brand color palette
  • Reimagine the layout from the reference
  • Add a new original headline/tagline
  • Include supporting marketing text
  • Use modern typography and clean spacing
  • Make the design feel premium and original
  • Enhance composition, lighting, and visual quality

Style direction:

Minimal, modern, eye-catching, Instagram-ready. Output should feel like a fresh premium brand post, not a remake.

Step 03

Generate the Video Ad

After generating the branding images and product visuals, ask ChatGPT to create a professional cinematic Seedance 2.0 prompt. Then use CapCut AI + Seedance 2.0 to generate the final ad video.

CapCut AI Seedance 2.0
Fast-Paced Brand Ad Prompt — For Seedance 2.0

Create a fast-paced, modern 15-second advertisement for [YOUR BRAND NAME]. Use dynamic cinematic shots with strong camera movement. Each shot should last 2–3 seconds, with smooth transitions and high visual energy.

Scene Direction:

  1. Start with the logo appearing in soft natural light
  2. Transition to product packaging placed on a clean natural surface
  3. Close-up of product texture with floating particles
  4. Show product being used or demonstrated
  5. Dynamic product shots from multiple angles
  6. Add subtle organic motion and brand elements
  7. Show text overlays with key brand messages
  8. Final hero shot with logo + tagline

Visual Style:

  • [Your brand color palette]
  • Soft warm lighting
  • Premium organic aesthetic
  • Consistent product design

Motion Style:

  • Smooth cinematic movement
  • Fast but clean transitions
  • Slight slow motion for premium feel

Avoid: no people, no clutter, no style changes between scenes.

High-End Product Ad Prompt — Alternative Version

Create a 15-second high-end commercial advertisement for [YOUR BRAND], using the provided product image as the main subject.

Editing Style:

Fast-paced cuts · Whip transitions · Zoom cuts · Match cuts · Light flashes · Smooth premium motion

Scene Breakdown:

  1. Dramatic logo reveal
  2. Product angle close-ups
  3. Macro product shot
  4. Usage shot
  5. Rotating product showcase
  6. Feature highlights
  7. Final hero shot

Final feel: Luxury · Premium · Modern · Original
